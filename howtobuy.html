How to Buy AZC

  1. Install MetaMask and connect to BNB Smart Chain (BEP-20).
  2. Fund your wallet with BNB for gas & swap.
  3. Open PancakeSwap V3 and connect wallet, or use this swap link: https://pancakeswap.finance/swap?outputCurrency=0xB9697BFC6eFA3e4893d0B1Ee75838eBE35AcedAd
  4. Approve token if prompted, set slippage (start with 1% or 2%), then swap.
  5. Add the token to MetaMask using the contract address: 0xB9697BFC6eFA3e4893d0B1Ee75838eBE35AcedAd
